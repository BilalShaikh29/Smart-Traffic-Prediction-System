# Smart Traffic Prediction System 🚦

Machine learning based traffic prediction web application developed using Python, Flask and Random Forest.

## Features

- User Registration and Login
- Interactive Dashboard
- Traffic Prediction
- Traffic Status Detection
- Responsive UI

## Technologies Used

- Python
- Flask
- Scikit-Learn
- HTML/CSS
- Bootstrap
- SQLite

## Live Demo

https://smart-traffic-prediction-system-hl1u.onrender.com
